แผนภาพที่ใช้แสดงลำดับขั้นตอนการใช้งานระบบจากมุมมอง
ของผู้ใช้ โดยอธิบายเส้นทางที่ผู้ใช้เดินผ่านในระบบ ตั้งแต่เริ่มต้นใช้งาน ไปจนถึงการทำกิจกรรมต่าง ๆ ภายในระบบและสิ้นสุดการใช้งาน จะช่วยให้เห็นว่า ผู้ใช้ต้องผ่านหน้าจอหรือขั้นตอนใดบ้าง เพื่อให้บรรลุเป้าหมายหนึ่ง ๆ เช่น การเข้าสู่ระบบ การดูสินค้า หรือการเพิ่มสินค้าในรายการโปรด โดยแผนภาพจะแสดงลำดับการทำงานด้วยลูกศรที่เชื่อมโยงแต่ละขั้นตอนเข้าด้วยกัน ทำให้สามารถเข้าใจโครงสร้างการใช้งานของระบบได้อย่างชัดเจน
